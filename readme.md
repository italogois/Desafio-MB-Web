# Desafio MB Web

## Requerimentos

- Node.js (LTS)

## API

Seguir os passos abaixo para iniciar a API.

- Acessar o diretório `api`
- Executar o comando `npm install`
- Executar o comando `ndoe server.js`

## Frontend

Seguir os passos abaixo para iniciar a API.

- Acessar o diretório `frontend`
- Executar o comando `npm install`
- Executar o comando `npm run dev`
